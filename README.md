# bank_statement_dataset
Automated pipeline to consolidate and normalize multiple bank statements into a structured dataset for quick reconciliation and support

## 📌 Overview
This project is a data pipeline designed to process, clean, and consolidate multiple bank statements into a unified and structured dataset.

It enables efficient bank reconciliation by transforming raw financial data from different sources into a standardized format ready for analysis and matching.

---

## 🎯 Objective
- Ingest multiple bank statements (PDF, JPG, PNG, Excel, CSV, TXT)
- Standardize inconsistent formats and structures
- Build a unified dataframe with all transactions
- Prepare data for reconciliation and analysis

---

## ⚙️ Tech Stack
- Python
- Pandas
- File processing (PDF, JPEG, PNG, Excel, CSV, TXT)
- (Optional) Regex / custom parsing logic

---

## 🧠 Key Features
- 🔹 Multi-source ingestion (different banks / formats)
- 🔹 Data cleaning and normalization
- 🔹 Column standardization across files
- 🔹 Transaction-level consolidation
- 🔹 Scalable structure for reconciliation workflows
